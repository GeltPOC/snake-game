# Snake Game

Clásico juego Snake construido con Next.js, TypeScript y Canvas HTML5.

## Características
- Controles con teclado (flechas y WASD)
- Botones táctiles para móvil
- Dificultad creciente (velocidad aumenta cada 5 puntos)
- Récord guardado en localStorage
- Estilo retro con canvas HTML5

## Instalación

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
